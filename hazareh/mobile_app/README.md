# اپلیکیشن موبایل هنرستان هزاره صنعت

اسکلت اولیه اپلیکیشن Flutter که با API نسخه ۱ پروژه Laravel (مسیر `/api/v1/...`) ارتباط برقرار می‌کند.

## اجرا
```
flutter pub get
flutter run
```

پیش از اجرا، آدرس API را در `lib/services/api_service.dart` (متغیر `baseUrl`) با آدرس واقعی هاست خود جایگزین کنید.

## ساختار
- `lib/services/api_service.dart` — تمام فراخوانی‌های REST API (اخبار، ثبت‌نام، مشاوره، همایش، احراز هویت)
- `lib/screens/` — صفحات اصلی اپ (خانه، اخبار، ورود، پروفایل، پیش‌ثبت‌نام)
- `lib/models/` — مدل‌های Dart معادل پاسخ‌های API
- `lib/theme/` — رنگ‌بندی هماهنگ با هویت بصری سایت (آبی صنعتی + کهربایی)

## وضعیت
این یک **اسکلت پایه (starter scaffold)** است که معماری، اتصال API، و صفحات اصلی را پیاده‌سازی می‌کند.
برای انتشار نهایی نیاز به تکمیل تمام صفحات، آیکن/اسپلش اسکرین اختصاصی، تست روی دستگاه واقعی،
و بیلد نسخه release (`flutter build apk` / `flutter build ios`) خواهد بود.
